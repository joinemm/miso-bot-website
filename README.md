# miso-bot-website

Website for [Miso Bot](https://github.com/joinemm/miso-bot)

Styling provided by [Bulma](https://bulma.io/)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
